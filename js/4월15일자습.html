할일을 입력하세요.

undefined: 값을 할당하지 않으면 undefined임. NaN: undefined와 숫자 연산하면 NaN 나옴. Infinity: 0으로 나누면 Infinity 나옴. -Infinity: 음수를 0으로 나누면 -Infinity 나옴. Number.MAX_SAFE_INTEGER: 안전한 최대 정수, 그 이상은 정확도 문제 발생. =: 값을 변수에 할당. +=: 더한 값을 변수에 할당. -=: 뺀 값을 변수에 할당. *=: 곱한 값을 변수에 할당. /=: 나눈 값을 변수에 할당. %=: 나머지를 변수에 할당. 삼항 연산자는 조건에 따라 값을 선택하는 방법임. 형식: 조건 ? 참일 때 값 : 거짓일 때 값 조건이 true이면 첫 번째 값을, false이면 두 번째 값을 반환함. Short-circuit && (AND) 왼쪽과 오른쪽 모두 true일 때만 true, 하나라도 false면 false. 하나라도 false면 그 즉시 false를 반환함 (short-circuit). || (OR) 왼쪽이나 오른쪽이 하나라도 true이면 true, 둘 다 false일 때만 false. 하나라도 true면 그 즉시 true를 반환함 (short-circuit). Falsy 값: false, 0, "", null, undefined, NaN (이 값들은 false처럼 취급). Truthy 값: Falsy가 아닌 값들 (예: true, 1, "non-empty string"). ||: 첫 번째 truthy 값 반환. &&: 첫 번째 falsy 값 반환. ===: 값과 타입이 같아야 true 반환. Number(): 문자열을 숫자로 변환. + "": 숫자를 문자열로 변환. <, >, <=, >=: 숫자로 변환해서 비교. ===: 값과 타입이 같아야 true. !==: 값이나 타입이 다르면 true. ==, !=: 타입 변환 후 비교.
